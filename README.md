VYBE Prototype
=============

What you have:
- index.html : main single-page prototype (splash screen + feed UI + create modal)
- styles.css : styling with the purple->blue->pink gradient palette
- app.js : minimal interactivity (splash fade, like toggle, publish client-only posts)
- assets/logo.png : the generated logo image used for the splash and header

How to open:
1. Download the ZIP file included (vybe_prototype.zip).
2. Unzip and open index.html in your browser (double-click works).
3. The splash screen will appear, then transition into the feed. Use the + create button to publish a test post.
